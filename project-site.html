Personal Brand Site

Process

The Idea

I wanted a personal site that reflected how I actually work — not a template dropped into WordPress, but something built from the ground up using the same AI-assisted workflow I use professionally. The site itself became a proof of concept for iterative development with Claude as a coding partner.

Design Direction

Dark editorial aesthetic with an emphasis on typography and negative space. Inspired by 21st.dev component patterns and modern portfolio sites. Every section was designed to feel intentional — clean enough to let the content breathe, but with enough visual personality to stand apart from cookie-cutter portfolios.

AI-Assisted Development

The entire site was built through conversation with Claude in Cowork mode. Every line of HTML, CSS, and JavaScript was written collaboratively — from the CSS design token system, to the Three.js particle animation, to the interactive button hover states. No boilerplate generators, no frameworks, just vanilla code shaped through natural language iteration.

Technical Highlights

Three.js dotted-surface hero with wave animation. Spline 3D robot that doubles as a chatbot trigger. CSS custom property design system for consistent theming. Scroll-reveal animations, mouse-tracking gradient effects, and multi-layer conic gradient borders on the chat widget. Everything is static HTML/CSS/JS — zero dependencies, zero build step.

What I Learned

Building a full site through AI pair-programming taught me how to think in terms of clear, composable instructions. I got better at describing visual intent in words, debugging layout issues conversationally, and knowing when to let the AI lead vs. when to steer. It also reinforced that AI tooling doesn't replace taste — it amplifies it.
